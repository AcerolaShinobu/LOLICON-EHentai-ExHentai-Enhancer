# LOLICON E-Hentai / ExHentai Enhancer

[English](README.en.md) | **简体中文**

自用油猴脚本
自动适配窗口尺寸、缩略图调整（大小/间距/列数）、快捷收藏、无限滚动、加载更多缩略图、快捷标签 & 搜索增强、标签自动补全、缩略图预览

## 获取

油猴脚本:
[Greasy Fork](https://greasyfork.org/scripts/516145)
[Sleazy Fork](https://sleazyfork.org/scripts/516145)

## 功能介绍

### 布局调整

- **布局控制模式** - 控制缩略图布局的核心适配逻辑，支持**缩放优先**或**列数优先**（支持在列表页和画廊页独立设置）
- **缩放优先** - 保持设定缩放倍率，列数随可用宽度自动变化，支持**自动**、**限制列数**、**限制宽度**
- **列数优先** - 保持固定列数，缩略图随窗口宽度自动缩放，可限制最大/最小**缩放倍率**
- **方形缩略图** - 将缩略图裁剪为统一正方形比例
- **缩略图边距/间距** - 调整缩略图之间及周围的间距
- **页面外边距/内边距** - 调整页面容器与窗口及内容之间的间距
- **全宽布局** - 允许页面扩展至最大可用宽度

### 缩略图预览

- **缩略图预览模式** - 鼠标悬停或使用快捷键时显示放大预览。支持 **关闭** / **悬停** / **快捷键** / **悬停 + 快捷键**（支持在列表页和画廊页独立设置）
- **悬停延迟** - 鼠标悬停后触发预览的等待时间
- **预览快捷键** - 选择触发预览的快捷键（Ctrl 或 Shift），可选 **松开按键时自动关闭预览**
- **移出预览框自动关闭** - 鼠标离开预览框范围时自动关闭预览
- **初始预览尺寸** - 设置悬停预览的尺寸计算逻辑。支持 **固定缩放** 或 **适应屏幕**
- **固定缩放** - 按设定 **缩放倍率** 显示预览，可选 **缩放限制** 限制预览最大显示尺寸
- **适应屏幕** - 根据窗口大小自动计算最佳预览尺寸，**显示方式** 支持 **完整显示** 或 **铺满窗口**
- **自动加载大图** - 开启后预览打开时自动加载高清大图替换模糊缩略图（注意：会消耗图片查看配额）
- **键盘快捷操作** - 预览框显示时, Space 键加载大图或循环切换预览尺寸, ← → ↑ ↓方向键切换至相邻缩略图, 按住 Space 键切换缩略图时加载大图

### 搜索 & 标签

- **快捷标签** - 保存常用搜索标签，点击即可快速填入搜索栏，支持分组管理
  - **交互操作**
    - 左键标签：添加/移除搜索标签，滚轮循环切换匹配模式（或 ~ 橙色 / 排除 - 红色），拖动排序
    - 左键 `[ + ]` 按钮：打开添加面板（标签：填写名称和语法，分组：仅填写名称），`[ ↵ ]` 按钮插入换行分隔
    - 右键任意按钮：打开标签编辑器并定位到对应行，可直接按格式编辑全部标签，实时同步，保存生效，取消还原
  - **分组系统**：点击分组导航栏可切换不同标签组，支持跨组拖拽排序（将标签拖入分组标题即可归类）
  - **数据格式**：`[tag]` / `[name @ tag]` / `[name @ tag1 tag2 ...]`，使用 `[# name]` 创建分组，使用 `[ _ ]`（下划线）插入换行分隔
  - [搜索指南](https://ehwiki.org/wiki/Gallery_Searching/Chinese)
- **画廊标签搜索** - 在画廊页点击标签即可自动弹出搜索栏并填入该标签，实现快速搜索
  - 支持左键添加或移除标签，滚轮循环切换匹配模式（或 ~ 橙色 / 排除 - 红色）
- **标签自动补全** - 在搜索栏、图库标签添加框、快捷标签添加框中提供自动补全建议
  - 按住 Ctrl/Cmd 点击候选项可插入为排除标签（添加 - 前缀）
  - 数据来源 [EhTagTranslation](https://github.com/EhTagTranslation/Database) 标签数据库，首次使用需下载约 6MB，与 EhSyringe 共存时自动检测避让
- **搜索增强** - 在搜索框内双击选中完整标签，搜索按钮支持右键/Ctrl+左键在新标签页打开，清除按钮仅清空输入内容，不会刷新页面
- **画廊分类增强** - 右键快速单选当前分类（自动取消其他分类）

### 浏览增强

- **无限滚动（列表页）** - 在搜索结果或列表页滚动到底部时自动加载下一页
- **更多缩略图（画廊页）** - 在画廊页滚动时自动加载后续页面的缩略图
  - 以上两项均可设置最大自动加载页数（默认：0 = 无限）
  - 支持自动跟随新内容连续加载，并可调整自动加载延迟
  - 达到上限后仍可手动继续加载，同时支持手动加载上一页
- **快捷收藏** - 无需打开弹窗即可快速收藏，支持多种布局及尺寸，支持点击或悬停展开
- **缩略图独立滚动** - 将画廊页缩略图区域设为独立滚动容器，在查看大量缩略图时保持页面结构稳定

### 其他功能

- **显示序号** - 在画廊标题前显示序号
- **实时更新网址** - 根据搜索结果页的浏览进度实时更新地址栏中的 URL，方便用户使用书签保存当前浏览位置
- **悬停显示完整标题** - 鼠标悬停在截断的画廊标题上时，显示完整标题内容
- **EH/ExH 切换按钮** - 在页面顶部显示切换按钮，点击即可在 E-Hentai 与 ExHentai 之间快速跳转

### 高级设置

- **网络请求间隔** - 控制连续 HTTP 请求的**起始发送间隔**
  - 请求过于频繁可能触发站点限制
- **网络请求并发数** - 控制允许同时执行的最大 HTTP 请求数量
  - 请求过于频繁可能触发站点限制
- **日志等级** - 控制控制台日志输出详细程度，用于调试和问题排查
- **语言** - 脚本界面语言（自动 / English / 简体中文 / 繁体中文 / 日本語 / 한국어 / Русский）
- **配置管理** - 导出设置备份，还原设置备份，重置设置为初始值，关闭所有开关
